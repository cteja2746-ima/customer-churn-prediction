# Customer Churn Prediction

## Project Objective
This project predicts customer churn using Machine Learning models and analyzes the important factors affecting customer retention.

----------------------------------

## Technologies Used
- Python
- PostgreSQL
- Pandas
- Scikit-learn
- XGBoost
- SHAP
- FastAPI

---------------------------------

## Machine Learning Models
- Logistic Regression
- Random Forest
- XGBoost

-----------------------------------

## Features
- Data preprocessing
- Feature engineering
- Model evaluation
- SHAP feature importance
- Confusion matrix visualization
- Accuracy comparison charts
- FastAPI integration

---------------------------------

## Project Workflow
1. Data Ingestion
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. API Development
